Virtual DOM

1. The entire structure of a web page can be represented using the DOM (Document Object Model)
2. JavaScript is used to communicate with the DOM
3. The virtual DOM represents a lightweight copy of the regular DOM
4. If the application needs to know information about the state of the elements, then the virtual DOM is accessed.
5. If you need to change the elements of the web page, the changes are first made to the virtual DOM
